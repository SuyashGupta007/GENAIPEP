# Chapter 1: What is Generative AI?

Generative AI (GenAI) is a branch of Artificial Intelligence that creates new content rather than simply classifying or predicting.

Traditional AI answers:

```text
Input Image → Cat or Dog?
```

Generative AI answers:

```text
Generate a new image of a dog wearing sunglasses.
```

Examples:

* Text Generation (ChatGPT)
* Image Generation (DALL-E)
* Video Generation (Sora)
* Music Generation
* Code Generation
* Synthetic Data Generation

# Machine Learning

Machine Learning enables systems to learn patterns from data.

Instead of:

```python
if age > 18:
    approve()
```

The model learns:

```text
Input Data
      ↓
Learning
      ↓
Prediction
```

Types:

## Supervised Learning

Uses labeled data.

Example:

```text
Email → Spam
Email → Not Spam
```

Algorithms:

* Linear Regression
* Logistic Regression
* Random Forest
* XGBoost

---

## Unsupervised Learning

No labels.

Example:

```text
Customer Segmentation
```

Algorithms:

* K-Means
* DBSCAN
* PCA

---

## Reinforcement Learning

Learning through rewards.

Example:

```text
Game Playing
Robotics
```

# Neural Networks

Inspired by biological neurons.

Neuron:

```text
Inputs
   ↓
Weighted Sum
   ↓
Activation Function
   ↓
Output
```

Formula:

```text
y = f(wx + b)
```

where:

```text
w = weight
x = input
b = bias
f = activation
```

---

# Activation Functions

## Sigmoid

```text
0 → 1
```

Used historically.

---

## Tanh

```text
-1 → 1
```

Better than sigmoid.

---

## ReLU

Most common.

```text
max(0,x)
```

Advantages:

* Fast
* Avoids vanishing gradients

---

# Backpropagation

Core learning mechanism.

Process:

```text
Prediction
     ↓
Calculate Error
     ↓
Propagate Error Backward
     ↓
Update Weights
```

---

# Gradient Descent

Optimization algorithm.

Formula:

```text
New Weight
=
Old Weight
-
Learning Rate × Gradient
```

---

# Why RNNs Failed

Before transformers:

```text
Word1 → Word2 → Word3 → Word4
```

Problems:

* Slow
* Long-term memory issues
* Not parallelizable

---

# Attention Is All You Need

Published in 2017.

Changed AI forever.

Core Idea:

```text
Every word can look at every other word.
```

---

# Self Attention

Sentence:

```text
The animal didn't cross the road because it was tired.
```

What does "it" refer to?

Transformer learns:

```text
it → animal
```

using attention.

---

# Query Key Value

Each token creates:

```text
Query
Key
Value
```

Attention Formula:

```text
Attention(Q,K,V)
=
softmax(QKᵀ/√d)V
```

This is arguably the most important equation in modern AI.

---

# Multi Head Attention

Instead of one attention mechanism:

```text
Head1 → Grammar
Head2 → Context
Head3 → Meaning
Head4 → Relationships
```

---

# Positional Encoding

Transformers don't know order naturally.

Need:

```text
Position 1
Position 2
Position 3
```

Positional encoding provides this information.

# Chapter 4: Embeddings

Embeddings convert text into vectors.

Example:

```text
Dog
```

becomes:

```text
[0.123, -0.77, 0.92 ...]
```

---

# Why Embeddings Matter

They capture meaning.

Example:

```text
Dog
Puppy
```

Vectors close together.

```text
Dog
Car
```

Vectors far apart.

Most common metric.


# Chapter 6: RAG

Retrieval Augmented Generation

Most important enterprise GenAI concept.

---

# Problem

LLMs don't know your PDFs.

Example:

```text
Company Handbook.pdf
```

GPT has never seen it.

---

# Solution

RAG

Pipeline:

```text
PDF
 ↓
Chunking
 ↓
Embeddings
 ↓
Vector DB
 ↓
Retriever
 ↓
LLM
 ↓
Answer
```

---

# Chunking

Breaking documents.

Example:

```text
500 characters
100 overlap
```

---

# Retrieval

Find relevant chunks.

Example:

```text
"What is leave policy?"
```

Retrieve:

```text
Leave Policy Section
```

---

# Why RAG Beats Fine Tuning

Advantages:

```text
Cheap
Fast
Updatable
No Retraining
```
